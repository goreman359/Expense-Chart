# Описание программы :


Expense Chart — консольное приложение для учёта личных расходов.
Позволяет добавлять, просматривать, удалять записи (сумма, категория, дата), фильтровать их по категории или периоду, подсчитывать общую сумму за выбранный промежуток времени и строить столбчатую диаграмму расходов по категориям с помощью matplotlib. Данные сохраняются в файл expenses.json и загружаются при следующем запуске.




# Возможности программы «Expense Chart»:

-обавление расходов с проверкой суммы (положительное число), категории (непустая) и даты (формат ГГГГ-ММ-ДД).

-росмотр полного списка расходов с нумерацией.

-даление любой записи по её номеру.

-ильтрация расходов по заданной категории.

-ильтрация расходов по периоду (начальная и конечная даты) с автоматическим подсчётом общей суммы.

-изуализация расходов — столбчатая диаграмма сумм по категориям (через matplotlib).

-втоматическое сохранение всех записей в файл expenses.json при выходе.

-агрузка ранее сохранённых данных при повторном запуске.

# Как запустить?:
Убедитесь, что установлен Python (версия 3.8 или выше).
Проверить можно командой:
python --version

Установите библиотеку matplotlib, если она ещё не установлена:
pip install matplotlib

Скачайте файл программы (например, expense_chart.py) в любую папку.

Откройте терминал (командную строку) в этой папке и выполните:
python expense_chart.py
